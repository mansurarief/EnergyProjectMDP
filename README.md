# EnergyProjectMDP

## Pushing changes to Github

1. Git status

```bash
git status
```

2. Add the files that you have changed  (dot means all)
```bash
git add .
```

3. Commit the change with a proper message
```bash
git commit -m "Put your message"
```

4. Push (upload) the change
```bash
git push
```
